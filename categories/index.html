HTML5 是定义 HTML 标准的最新的版本。 该术语表示两个不同的概念：

  • 它是一个新版本的 HTML 语言，具有新的元素，属性和行为。
  • 它有更大的技术集，允许更多样化和强大的网站和应用程序。这个集合通常缩写为 HTML5。
